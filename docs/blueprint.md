# **App Name**: SevaSetu

## Core Features:

- User Authentication & Profiles: Enable secure user sign-up and login, along with features for creating and managing profiles tailored for offering or seeking aid.
- Aid Posting & Management: Allow users to easily submit new help requests or volunteer offers, and manage their existing listings (edit, delete, view status).
- Search & Filtering: Provide robust search and filtering capabilities to help users find relevant help requests or available aid providers based on various criteria like location, category, and urgency.
- AI Request Categorization Tool: An AI tool that automatically analyzes newly posted help requests and suggests appropriate categories (e.g., medical, education, food, housing) to improve organization and search accuracy.
- Direct Messaging System: Implement a private, in-app messaging feature to facilitate secure and direct communication between individuals offering and seeking aid for coordination.
- Administrative Panel: A dedicated dashboard for NGO administrators to oversee all listings, manage user accounts, and review communication, ensuring safety and platform integrity.

## Style Guidelines:

- Primary Color: A bold, impactful red (#B52119) to convey passion, urgency, and a strong sense of community support.
- Background Color: A very dark, subtle reddish-gray (#282120) provides a sophisticated and strong foundation, allowing other elements to stand out.
- Accent Color: A vibrant berry pink (#D93B80) to add a touch of warmth and highlight interactive elements, creating strong contrast against the primary and background tones.
- Text Colors: Predominantly white text for maximum readability on the dark background, with black used sparingly for emphasis or on lighter UI elements.
- Headlines: 'Alegreya' (serif) for an elegant, intellectual, and humanistic touch, evoking trust and gravitas. Body Text: 'PT Sans' (sans-serif) for its modern, warm, and highly readable qualities suitable for longer content blocks.
- Clean, contemporary line-art icons should be used to represent various aid categories and platform functionalities, maintaining a consistent and clear visual language.
- A streamlined, user-friendly layout with ample white space (or dark space in this case) to ensure clarity and focus. Key actions and information should be prominently displayed for quick access.
- Subtle and functional animations, such as smooth transitions for content loading and clear feedback on interactive elements, to enhance user experience without being distracting.
